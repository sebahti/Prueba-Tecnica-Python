# Prueba-Tecnica-Python

## Lenguaje de programación y librerías

Este proyecto esta desarrollado en Python y SQLITE, la conexión entre estos fue realizaba bajo la librería sqlite3

## Requerimientos

1. VS Code 1.87.2 o superior
2. Python 3.9.12 o superior
4. Extensión de Python para Visual Studio Code
5. Virtualenv 

### Librerias

Para hacer la instalación de las librerías seguiremos los siquientes pasos:

Nota: Es necesario que Visual Studio Code esté abierto sobre la carpeta raiz del proyecto (Prueba-Tecnica-Python)

    1. Abrimos una nueva terminal
    2. Instalamos VirtualEnvc ejecuntando el siguiente comando en la terminal
        pip install virtualenv

       Creamos el entorno virtual:
       python -m venv env     

    3. Activamos el entorno virtual con el siguiente comando en la terminal
        env\Scripts\activate
        
    4. Instalamos las librerías desde la lista de requirements así con el siguiente comando
        pip install -r requirements.txt

EJECUCIÓN:
Posicionese desde el archivo Main.py que se encuentra en src y ejecute el run and debug (CTRL+SHIFT+D o tecla F5)
